## 本地阅读

如果你想要离线阅读该内容，需要克隆仓库并进行编译。

需求：

* R
* bookdown

编译：

```
# 克隆
git clone https://github.com/ShixiangWang/py4ds-CN.git

# 点击目录下的bookdown-chinese.Rproj文件（如果安装了RStudio）
# 如果没有，请将仓库切换为工作目录

# 在R控制台运行
bookdown::render_book("index.Rmd", "bookdown::gitbook")
```
